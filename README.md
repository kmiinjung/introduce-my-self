# introduce-my-self
글의 구조를 이해하고, 영상의 문법으로 옮기는 OOO입니다.  국어국문학을 전공하며 좋은 이야기의 뼈대를 세우는 법을 배웠고, 미디어 커뮤니케이션을 통해 그 이야기를 시각화하는 즐거움을 알아가고 있습니다. 1년간의 편의점 아르바이트와 매주 거르지 않은 봉사활동으로 다져진 끈기는, 긴 호흡이 필요한 영상 제작 현장에서 저를 지탱하는 가장 큰 힘입니다. 현재는 수업에서 배운 촬영과 편집 기술을 바탕으로 영상 공모전에 몰입하며, 텍스트 너머의 생생한 이미지를 구현하는 실전 감각을 익히고 있습니다. 기획부터 편집까지, 현장의 동료들과 유연하게 소통하며 끝까지 완성도 있는 결과물을 만들어내겠습니다.
