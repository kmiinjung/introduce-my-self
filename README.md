# kiminjung
김민정의 자기소개

**서사의 뼈대 위에 시각적 전율을 입히는 연출가, 김민정**

제가 국어국문학을 전공하며 제가 가장 애착을 가진 영역은 현대문학입니다. 파편화된 내면과 복잡한 사회 현실을 담기 위해 전통적 구조를 해체하는 현대소설의 전위적인 모습은 제게 큰 충격이었습니다. 저는 텍스트 속에 갇혀 상상 속에만 머물던 소설 속의 장면들을 영화의 렌즈를 통해 생생하게 시각화하는 과정에 흥미를 느껴 영화 연출에 몰입하여 국어국문학에 더불어 미디어커뮤니케이션 영상콘텐츠 학부를 전공하고 있습니다. 

특히 소설 속 인물의 내면 심리를 묘사하는 집요한 문장들을 카메라의 구도와 조명, 편집의 호흡으로 치환하는 과정에 몰입하고 있습니다. 종이 위에서 명멸하는 추상적인 상상이 스크린이라는 실체를 얻고, 그 미장센이 관객의 감각에 직접 닿을 때의 전율은 제가 영화 연출에 뛰어들게 된 결정적인 계기가 되었습니다. 단순한 시각화를 넘어, 텍스트가 품은 서사의 온도를 화면 위로 오롯이 옮겨내는 것이 저의 연출적 지향점입니다. 

매주 이어온 봉사활동은 타인의 삶에 깊게 주파수를 맞추는 법을 가르쳐주었고, 이는 현장에서 스태프들과 뜨겁게 호흡하는 유연한 태도가 되었습니다. 현재 전공과목인 '촬영과편집' 수업에서 영상 공모전을 준비하며 기획부터 이어질 후반 작업까지 영상 제작의 전 과정을 실전으로 체득하며 많은 경험을 쌓고 있습니다. 상상의 파편들을 생생한 화면으로 응집해내는 책임감 있는 연출가가 되어, 저는 영화 산업의 한 현장에서 한 편의 작품을 처음부터 끝까지 책임지고 이끄는 전문가로 성장해 있을 것입니다.
