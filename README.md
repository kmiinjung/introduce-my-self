# 자기소개
김민정의 자기소개

**텍스트에서 스크린으로 : 인문학적 상상의 시각화를 다루는 김민정**

문학적 상상력을 영상 언어로 구현하는 연출가를 지향하는 김민정입니다. 
국어국문학을 전공하며 접한 현대문학 작품들은 제게 단순한 텍스트를 넘어 이미지와 공간으로 확장되었습니다. 
특히 <소나기>와 <동백꽃>에서 느껴지는 따뜻하면서도 처연한 정서는 머릿속에서 장면처럼 구체화되었고, 이를 실제 영상으로 구현하고자 하는 열망은 미디어커뮤니케이션 영상콘텐츠 전공으로 이어졌습니다.

영상 제작을 배우는 과정에서 문학적 서사를 시각적 이미지로 번역하는 능력이 저의 강점임을 깨달았습니다. 
현재 ‘촬영과 편집’ 수업에서 ‘무모함’을 주제로 단편영화를 제작하며, 소설 <난장이가 쏘아올린 작은 공>의 서사처럼 현실을 초월하려는 인간의 희망을 카메라 구도와 화면 구성, 편집 리듬을 통해 시각적 은유로 표현하고 있습니다.

이처럼 문학적 이미지를 영상 언어로 치환하는 과정 속에서 연출에 대한 확신을 쌓아가고 있습니다. 앞으로도 인문학적 통찰과 제작 역량을 결합하여 보이지 않는 감정과 서사를 설득력 있는 이미지로 구현하고, 작품의 메시지를 끝까지 책임지는 연출가로 성장하고자 합니다.
