# kiminjung
김민정의 자기소개

이야기의 뼈대를 세우고, 이를 영상 언어로 풀어내는 김민정입니다.

국어국문학을 전공하며 좋은 서사의 구조를 분석하는 법을 배웠고, 이를 미디어 커뮤니케이션이라는 렌즈를 통해 시각화하는 과정에 몰입하고 있습니다. 매주 꾸준히 이어온 봉사 활동은 타인과 깊게 공감하고 소통하는 법을 가르쳐주었으며, 이는 현장에서 스태프들과 호흡을 맞추는 저만의 유연한 태도가 되었습니다. 현재는 '촬영과 편집'수업에서 공모전을 준비하며 기획부터 후반 작업까지 영상 제작의 전 과정을 실전으로 익히고 있습니다. 글 속에 머물던 상상을 생생한 화면으로 구현해내는, 책임감 있는 연출가가 되겠습니다.
