# kiminjung
김민정의 자기소개

**텍스트에서 스크린으로 : 인문학적 상상의 시각화를 다루는 김민정**

국어국문학을 전공하며 접한 여러 현대문학 작품들은 제게 단순한 텍스트 이상의 영감으로 다가올 때가 많았습니다. 
대표적으로 황순원의 <소나기>나, 김유정의 <동백꽃> 처럼 따뜻하지만 한편으론 처연한 색채를 머금은 문장들은 제 머릿속에서 늘 살아 움직이는 이미지와 공간이 되었습니다. 
종이 위에서 명멸하는 이 추상적인 서사들을 영화의 렌즈를 통해 생생하게 시각화하고 싶다는 열망은 자연스럽게 미디어커뮤니케이션 영상콘텐츠 전공으로 이어졌고, 인문학적 깊이와 기술적 역량을 동시에 갖춘 연출가로 성장하는 밑거름을 쌓고 있습니다.
